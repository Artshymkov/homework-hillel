IndexError - помилка одержання елемента за індексом, для відтворення потрібно спробувати отримати елемент за індексом, якого немає у списку, приклад: ['Hello', 'world'][10]

KeyError - помилка ключа, для відтворення потрібно спробувати отримати значення за ключем, якого немає у словнику, приклад: my_dict = {'a': 1, 'b': 2}; print(my_dict['c'])

TypeError - помилка типу, для відтворення потрібно виконати операцію з об'єктом неправильного типу, приклад: x = 5 + '2'

NameError - помилка імені, для відтворення потрібно використати змінну, яка не була визначена, приклад: print(x)

FileNotFoundError - помилка файлу не знайдено, для відтворення потрібно відкрити файл, який не існує, приклад: file = open('nonexistent_file.txt', 'r')
